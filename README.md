# Rugrats React App

This is a simple React application that displays four Rugrats characters using a reusable `Image` component.

## Features

- Built with React
- Uses a reusable `Image` component
- Displays four cartoon character images
- Uses props for image source and character name
- Styled with CSS grid

## Characters Included

- Tommy
- Chuckie
- Angelica
- Susie

## How to Run

1. Open the project folder in VS Code
2. Open the terminal
3. Run:

```bash
npm install
npm run dev
